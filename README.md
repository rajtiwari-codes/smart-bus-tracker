# Smart Bus Tracker

SIH prototype for smart public transport: dynamic bus route tracking, live progress simulation, and delay visibility on interactive maps.

## Features
- Route search (start to destination)
- Live bus movement simulation
- Distance and duration display
- Delay estimation and reason
- Journey progress bar
- Step-by-step route instructions

## Tech Stack
- HTML
- Tailwind CSS
- JavaScript
- Leaflet.js
- OpenStreetMap
- OpenRouteService API

## Run Locally
1. Open project in Windsurf/VS Code
2. Run with Live Server
3. Open `promtEng.html`

## Project Status
Frontend prototype (no backend/database yet)
